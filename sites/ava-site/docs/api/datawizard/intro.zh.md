DataWizard 是一个 js/ts 的前端数据处理类库。在 AVA 的框架中，它被用来“理解”输入端的数据集。不过，它也可以独立地被用来开发一些数学统计、数据模拟之类的功能。


## 包

```bash
$ npm install @antv/dw-analyzer   // 数据分析
$ npm install @antv/dw-random     // 数据生成
$ npm install @antv/dw-transform  // 数据变换
```
