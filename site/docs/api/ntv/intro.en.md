`NarrativeTextVis` is a react component to render [ntv-schema](../../guide/ntv/ntv-schema.zh.md)。

## Props

| 属性         | 类型                | 描述                 | 默认值          |
| ------------ | ------------------- | -------------------- | --------------- |
| spec         | `NtvTypes.NarrativeTextSpec`             | data             | 无              |
| size         | 'default' \| 'small'    | font size, default is 14px, small is 12px           | 'default'              |
