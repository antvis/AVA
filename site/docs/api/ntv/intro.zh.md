`NarrativeTextVis` 是一个用于渲染数据解读文本 React 组件。

## Props

| 属性         | 类型                | 描述                 | 默认值          |
| ------------ | ------------------- | -------------------- | --------------- |
| spec         | `NtvTypes.NarrativeTextSpec`             | 结构数据             | 无              |
| size         | 'normal' \| 'small'    | 字体大小，normal 是 14px，small 是 12px           | 'default'              |
| theme         | 'light' \| 'dark'    | 主题色，当前支持 light 亮色模式和 dark 暗色模式          | 'light'              |
