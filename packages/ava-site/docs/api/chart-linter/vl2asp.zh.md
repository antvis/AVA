```sign
vl2asp(spec: VegaLite, fieldInfos: Field[]): string[]
```

### 参数

* **spec** * 图表配置
  * _必要参数_
  * `参数类型`: VegaLite

  * _必选参数_
  * `参数类型`: `'Field[]'`

### 返回值

*`string[]`*

将图表转换为 ASP 化的图表配置字符串。
